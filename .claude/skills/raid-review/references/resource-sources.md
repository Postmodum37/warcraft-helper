# Resource Sources

How to find video guides, diagrams, and written resources for specific raid mechanics.

## Trusted Sources

### 1. MythicTrap
- **URL:** mythictrap.com
- **Best for:** Interactive positioning diagrams, movement patterns, visual mechanic breakdowns
- **Search:** `site:mythictrap.com [boss name]`
- **Notes:** Covers current tier bosses with clickable phase diagrams. Best visual resource for spatial mechanics.

### 2. YouTube Guides
- **Best for:** Video walkthroughs with mechanic-specific timestamps
- **Preferred creators:** Hazelnuttygames, Stankie, Quazii, Dratnos, Growl (tanks), Yumytv (healers)
- **Search:** `[boss name] [mechanic name] guide heroic [current tier name]`
- **Notes:** Prefer videos with timestamps in the description. When linking, include the timestamp for the specific mechanic section (append `?t=XXX` to YouTube URLs). Prefer videos under 15 minutes for mechanic-specific content.

### 3. Wowhead
- **URL:** wowhead.com
- **Best for:** Comprehensive written guides with ability descriptions, diagrams, and strategy breakdowns
- **Search:** `site:wowhead.com [boss name] guide`
- **Notes:** Usually has both quick strategy and in-depth guides. Good for linking to specific ability descriptions.

### 4. Icy Veins
- **URL:** icy-veins.com
- **Best for:** Role-specific boss guides (separate tank/healer/DPS perspectives)
- **Search:** `site:icy-veins.com [boss name] [role] guide`
- **Notes:** Particularly useful when a mechanic affects different roles differently. Link the role-specific version.

### 5. r/CompetitiveWoW
- **URL:** reddit.com/r/CompetitiveWoW
- **Best for:** Niche tips, community-discovered tricks, discussion of specific mechanics
- **Search:** `site:reddit.com/r/CompetitiveWoW [boss name] [mechanic name]`
- **Notes:** Use as a supplement, not primary source. Good for "how are people actually handling this" context.

## Search Strategy

When a mechanic problem is identified (e.g., "Shadow Orb damage on Crown of the Cosmos"):

1. **Construct search query:** `[boss name] [mechanic/ability name] guide heroic` (e.g., "Crown of the Cosmos shadow orbs guide heroic")
2. **Add role if relevant:** If the issue is role-specific, add the role (e.g., "healer guide")
3. **Search the web** and evaluate results against quality signals below
4. **Pick 1-2 resources** — one visual (video/diagram) and optionally one written

## Quality Signals

- **Freshness:** Published after the raid tier launched. Reject PTR/beta guides if live guides exist.
- **Specificity:** Covers the exact mechanic, not just a full boss overview. Timestamped videos score highest.
- **Creator reputation:** Prefer established guide creators (listed above) over unknown channels.
- **Visual content:** Clips showing correct execution of a mechanic are more valuable than text descriptions.
- **View count:** Higher views generally indicate better quality, but freshness matters more.

## Fallback Behavior

If no mechanic-specific resource is found:
- Link the general boss guide from Wowhead or Icy Veins
- Note: "No specific guide found for [mechanic] — see the full boss guide"
- **Never fabricate or guess URLs** — only link resources you found via web search
